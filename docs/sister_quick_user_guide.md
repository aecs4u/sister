# SISTER (Agenzia delle Entrate) — Quick User Guide

> Guida operativa ai servizi catastali e ipotecari telematici.
> Aggiornata: aprile 2026.

---

## 1. Cos'e SISTER

SISTER e il portale telematico dell'Agenzia delle Entrate per accedere a:

| Servizio | Descrizione |
|----------|-------------|
| **Banche dati catastali** | Visure, ricerche per soggetto/immobile/indirizzo, elenco immobili |
| **Banche dati ipotecarie** | Ispezioni, note, titoli, registri di pubblicita immobiliare |
| **Presentazione documenti** | Docfa, Pregeo, Voltura, Modello Unico, planimetrie |
| **Amministrazione** | Gestione utenza, contatti, pagamenti, convenzioni |

---

## 2. Chi lo usa

| Categoria | Accesso tipico |
|-----------|---------------|
| Professionisti tecnici | Geometri, ingegneri, architetti |
| Notai e pubblici ufficiali | Con convenzione dedicata |
| Soggetti privati convenzionati | Aziende, studi legali, agenzie immobiliari |
| Enti / istituzioni | Comuni, enti pubblici |

Per la consultazione serve una **convenzione attiva** con l'Agenzia.

---

## 3. Come si entra

### Percorso passo-passo

1. Vai nell'**area riservata** dell'Agenzia delle Entrate
2. Autenticati con:
   - **SPID** (Sistema Pubblico di Identita Digitale)
   - **CIE** (Carta d'Identita Elettronica)
   - **CNS** (Carta Nazionale dei Servizi)
   - oppure credenziali **Entratel / Fisconline / Sister** (dove previsto)
3. Vai in **Servizi**
4. Apri **Fabbricati e terreni**
5. Se sei gia abilitato: entra direttamente nelle funzioni Sister
   Se non sei abilitato: seleziona **Adesione ai servizi Sister**

---

## 4. Prima attivazione

Se non sei ancora abilitato:

| Passaggio | Azione |
|-----------|--------|
| 1 | Accedi dall'area riservata |
| 2 | Apri **Servizi > Fabbricati e terreni > Adesione ai servizi Sister** |
| 3 | Scegli il tipo di convenzione (vedi sotto) |
| 4 | Compila la domanda |
| 5 | Firma digitalmente (dove richiesto) |
| 6 | Invia la richiesta |
| 7 | Controlla lo stato in **Stato della richiesta** |

### Tipi di convenzione

| Tipo | Destinatari |
|------|-------------|
| Consultazione persona fisica | Professionisti singoli, privati |
| Consultazione persona giuridica | Studi, societa, aziende |
| Comuni / enti | Enti pubblici, amministrazioni |
| Presentazione documenti | Soggetti abilitati a Docfa, Pregeo, Voltura |

## 5. Le aree principali

### A. Consultazioni catastali

Usale per ottenere informazioni dal catasto:

| Funzione | Quando usarla |
|----------|---------------|
| **Visura catastale** | Dati catastali, intestazioni, identificativi, rendita |
| **Ricerca per immobile** | Hai foglio / particella / subalterno |
| **Ricerca per soggetto** | Hai nome, CF o denominazione |
| **Ricerca per indirizzo** | Hai l'indirizzo dell'immobile |
| **Ricerca per partita** | Hai il numero di partita catastale |
| **Elenco immobili** | Lista completa per un dato foglio/particella |
| **Visura / ricerca nota** | Cerchi una specifica nota catastale |
| **Estratto di mappa** | Mappa catastale di una particella |
| **Punti fiduciali** | Coordinate dei punti di riferimento topografico |

### B. Ispezioni ipotecarie

Usale per accedere ai registri immobiliari:

| Funzione | Quando usarla |
|----------|---------------|
| **Ricerca per persona fisica** | CF di una persona naturale |
| **Ricerca per persona giuridica** | P.IVA / denominazione di una societa |
| **Ricerca per immobile** | Dati catastali dell'immobile |
| **Ricerca per nota** | Numero specifico di nota |

> Il sistema mostra il **costo** prima della conferma. Verifica sempre l'importo.

### C. Presentazione documenti

Area riservata ai soggetti abilitati:

| Documento | Scopo |
|-----------|-------|
| **Docfa** | Dichiarazione di nuova costruzione o variazione catastale |
| **Pregeo** | Atti di aggiornamento geometrico del catasto terreni |
| **Voltura** | Trasferimento di proprieta nei registri catastali |
| **Modello Unico** | Pratica unificata (notarile) |
| **Visura planimetrica** | Richiesta planimetria di un immobile |

### D. Gestione utenza

| Funzione | Descrizione |
|----------|-------------|
| Gestione utenti | Aggiungere/rimuovere utenti collegati alla convenzione |
| Dati e contatti | Aggiornare email, telefono, indirizzo |
| Abilitazioni | Gestire i permessi operativi per ogni utente |

### E. Pagamenti / gestione contabile

| Funzione | Descrizione |
|----------|-------------|
| Attivazione utenze | Prima attivazione o rinnovo |
| Ricevute e avvisi | Consultazione storico pagamenti |
| Gestione contabile | Funzioni amministrative collegate |

## 6. Flussi operativi rapidi

### Flusso 1 — Visura catastale per immobile

| Passo | Azione |
|-------|--------|
| 1 | Entra in **Consultazioni catastali** |
| 2 | Scegli **Ricerca per immobile** |
| 3 | Inserisci: Comune, sezione (se presente), foglio, particella, subalterno (se presente) |
| 4 | Invia la richiesta |
| 5 | Apri la risposta |
| 6 | Salva o stampa il risultato |

### Flusso 2 — Visura per persona

| Passo | Azione |
|-------|--------|
| 1 | Entra in **Consultazioni catastali** |
| 2 | Scegli **Persona fisica** o **Persona giuridica** |
| 3 | Inserisci i dati identificativi (CF / denominazione) |
| 4 | Scegli l'ambito: **nazionale**, **provinciale** o **comunale** |
| 5 | Invia |
| 6 | Apri l'elenco immobili / la visura restituita |

### Flusso 3 — Ispezione ipotecaria

| Passo | Azione |
|-------|--------|
| 1 | Entra in **Reparti di Pubblicita Immobiliare** / Ispezioni ipotecarie |
| 2 | Seleziona il tipo di ricerca: persona fisica, persona giuridica, immobile, nota |
| 3 | Inserisci i dati richiesti |
| 4 | Verifica il costo (mostrato dal sistema prima della conferma) |
| 5 | Conferma |
| 6 | Scarica la risposta e la ricevuta |

### Flusso 4 — Planimetria catastale

| Passo | Azione |
|-------|--------|
| 1 | Entra in **Presentazione documenti > Visura planimetrica** |
| 2 | Compila la richiesta indicando un soggetto intestatario |
| 3 | Firma digitalmente il modulo |
| 4 | Invia la richiesta |
| 5 | Controlla lo stato |
| 6 | Visualizza e salva la planimetria |

> **Attenzione:** la visura planimetrica e riservata a soggetti abilitati. Non viene rilasciata se:
> - il soggetto indicato non risulta intestatario;
> - il codice fiscale non e validato;
> - salvo casi particolari (es. incarico da un erede).

### Flusso 5 — Due diligence immobiliare completa

| Passo | Azione | Perche |
|-------|--------|--------|
| 1 | **Ricerca per persona** (nazionale) | Trova tutti gli immobili del soggetto in Italia |
| 2 | **Visura catastale** per ogni immobile | Dati catastali, rendita, intestazione |
| 3 | **Elenco immobili** per foglio/particella | Lista completa dei subalterni |
| 4 | **Ispezione ipotecaria** per soggetto | Formalita, ipoteche, pignoramenti |
| 5 | **Elenco note** per conservatoria | Dettaglio delle note ipotecarie |
| 6 | **Estratto di mappa** | Mappa catastale della particella |
| 7 | **Planimetria** (se abilitato) | Piantina dell'unita immobiliare |

## 7. Cosa preparare prima di usare SISTER

| Elemento | Quando serve |
|----------|-------------|
| SPID / CIE / CNS | Sempre (autenticazione) |
| Convenzione attiva | Sempre (verifica in Gestione utenza) |

| Codice fiscale del soggetto | Ricerche per persona, planimetrie |
| Delega / incarico | Quando si agisce per conto di terzi |
| Metodo di pagamento | Ispezioni ipotecarie, certificati |

## 8. Errori frequenti e soluzioni

| Errore | Causa | Soluzione |
|--------|-------|----------|
| Immobile non trovato | Riferimenti catastali incompleti o errati | Verifica comune, sezione, foglio, particella, sub |
| CF non validato | Codice fiscale errato o non attivo | Controlla il CF con il servizio di verifica dell'Agenzia |
| Planimetria non rilasciata | Soggetto non intestatario o CF non validato | Verifica intestazione e correttezza del CF |
| Planimetria non trovata | Menu sbagliato | Si trova in **Presentazione documenti**, non in Consultazioni |
| Omonimie nei risultati | Ricerca per nome troppo generica | Usa i dati catastali (foglio/particella) quando possibile |
| Funzione bloccata | Convenzione scaduta o profilo non abilitato | Controlla in Gestione utenza e Pagamenti |
| Visura vs ispezione confuse | Sono due servizi diversi | Catastale = dati immobile; ipotecaria = registri pubblici |
| Scrivania del Territorio | Non piu disponibile | Migrata su Sister dal 1 aprile 2026 |

## 9. Password, accesso e supporto

### Problemi di accesso

| Problema | Cosa fare |
|----------|----------|
| Password dimenticata | Procedura di recupero dall'area riservata |
| Contatti non aggiornati | Aggiorna in **Gestione utenza** prima del recupero |
| Convenzione scaduta | Rinnova tramite **Pagamenti / Gestione contabile** |
| Errore tecnico | Contatta l'assistenza Sister online |

### Assistenza

- **Assistenza Sister**: per problemi tecnici, convenzioni, password
- **Assistenza online servizi catastali e ipotecari**: per funzionamento dei servizi
- **Gestione utenza**: per controllare lo stato dell'utenza e i permessi
