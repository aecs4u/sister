# SISTER (Agenzia delle Entrate) — Cheat Sheet Operativo

> Riferimento rapido per l'uso quotidiano dei servizi catastali e ipotecari.
> Aggiornato: aprile 2026.

---

## Accesso

| Passaggio | Percorso |
|-----------|----------|
| **Login** | Area riservata (SPID / CIE / CNS) |
| **Navigazione** | Servizi > Fabbricati e terreni > Sister |

### Prima attivazione

Servizi > Fabbricati e terreni > **Adesione ai servizi Sister** > scegli convenzione > compila > firma digitale > invia > controlla in **Stato della richiesta**.

---

## Mappa dei menu principali

| Menu | A cosa serve |
|------|-------------|
| **Consultazioni e certificazioni** | Visure, ricerche catastali, estratti di mappa, punti fiduciali |
| **Ispezioni ipotecarie** | Registri immobiliari, note, titoli depositati |
| **Presentazione documenti** | Docfa, Pregeo, Voltura, Modello Unico, planimetrie |

## Quale ricerca scegliere

| Hai... | Usa... | Dove |
|--------|--------|------|
| Foglio / particella / sub | **Ricerca per immobile** | Consultazioni catastali |
| Nome / CF / denominazione | **Ricerca per persona** | Consultazioni catastali |
| Indirizzo | **Ricerca per indirizzo** | Consultazioni catastali |
| Numero di nota | **Ricerca nota** | Ispezioni ipotecarie |
| Serve il registro immobiliare | **Ispezione ipotecaria** | Reparti Pubblicita Immobiliare |
| Serve la planimetria | **Visura planimetrica** | Presentazione documenti |

## Flussi operativi rapidi

### Visura catastale per immobile

```
Comune > sezione (se c'e) > foglio > particella > subalterno
> invio > verifica risposta > salva / stampa
```

### Visura catastale per persona

```
Persona fisica / giuridica > CF o denominazione
> ambito (nazionale / provinciale / comunale) > invio > apri risultato
```

### Ispezione ipotecaria

```
Tipo ricerca (persona / immobile / nota)
> dati soggetto / immobile / nota > verifica costo > conferma
> scarica esito + ricevuta
```

### Planimetria catastale

```
Presentazione documenti > Visura planimetrica > compila richiesta
> firma digitale > invia > controlla stato > scarica
```

> **Nota:** la planimetria e riservata a soggetti abilitati; non viene rilasciata se il CF non e validato o il soggetto non risulta intestatario.

## Checklist pre-invio

Prima di cliccare **Invia**, verifica:

- [ ] Convenzione attiva
- [ ] Profilo corretto
- [ ] Dati inseriti completi
- [ ] Codice fiscale corretto e validato
- [ ] Eventuale intestatario corretto
- [ ] Firma digitale disponibile (se richiesta)
- [ ] Pagamento pronto (se dovuto)

---

## Quando serve la firma digitale

- Adesione / richieste di abilitazione
- Planimetrie (visura planimetrica)
- Presentazione documenti (Docfa, Pregeo, Voltura)
- Altri flussi contrassegnati dal sistema come firmabili

## Scorciatoie pratiche

| Scenario | Strategia |
|----------|-----------|
| **Due diligence immobiliare** | Prima visura catastale, poi ispezione ipotecaria |
| **Evitare omonimie** | Parti dai dati catastali (foglio/particella) invece che dal nome |
| **Indagine completa su soggetto** | Ricerca persona nazionale > poi per ogni provincia > ispezione ipotecaria |
| **Problemi ricorrenti** | Controlla password, convenzione, ruolo utente e contatti in Gestione utenza |
| **Costo ispezione** | Il sistema mostra il costo prima della conferma; verifica sempre |

## Costi indicativi

| Servizio | Costo |
|----------|-------|
| Visura catastale | Gratuita (per convenzionati) |
| Ispezione ipotecaria | A pagamento (varia per tipo) |
| Estratto di mappa | Gratuito (per convenzionati) |
| Planimetria | Gratuita (solo per soggetti abilitati) |
| Certificati catastali | A pagamento |

> I costi esatti dipendono dalla convenzione attiva. Il sistema mostra l'importo prima della conferma.

---

## Errori comuni

| Errore | Soluzione |
|--------|----------|
| Immobile non trovato | Verifica tutti i riferimenti catastali (comune, foglio, particella, sub) |
| CF non validato | Controlla che il codice fiscale sia corretto e attivo |
| Planimetria non disponibile | Verifica di essere in Presentazione documenti > Visura planimetrica |
| Funzione non disponibile | Verifica che la convenzione sia attiva e il profilo abilitato |
| Confusione visura / ispezione | La visura catastale e diversa dall'ispezione ipotecaria: dati diversi, menu diversi |
