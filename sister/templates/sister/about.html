About SISTER

What is SISTER?

SISTER is an automated cadastral data extraction service that queries the Italian Land Registry (Agenzia delle Entrate — SISTER Portal) via browser automation.

It provides:

  • REST API — FastAPI-based HTTP endpoints for programmatic access
  • CLI — sister command-line tool with 16+ query types
  • Web UI — This dashboard for interactive queries and result browsing

Available Search Types

Property Search (Immobili)
Owner Lookup (Intestati)
Person Search (Soggetto)
Company Search (Azienda)
Property List (Elenco)
Address Search
Partita Search
